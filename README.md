## ðŸ“˜ What Are JavaDocs?
JavaDocs are automatically-generated documentation pages created from special comment blocks inside Java code.  
They help developers understand how classes, methods, and objects behave.

JavaDoc comments begin with:

```java
/**
 * This is a JavaDoc comment.
 */
```
